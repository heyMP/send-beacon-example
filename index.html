You've Stumbled Upon a Web Dev Secret!

This page might look like a joke, but it's actually demonstrating some cool, modern browser APIs.

Ever wonder how websites know when you've abandoned them? Part of the magic is the 'visibilitychange' event. Your browser kindly tells us when you switch tabs, minimize the window, or navigate away. Don't believe me? Read the docs!

But what do we do with that power? We could just cry, but instead, we use the Beacon API. It lets us send a little packet of data back to our server right as you're leaving, without slowing down your escape. It's the perfect way to get in the last word. Check it out here: Beacon API docs.

So go ahead, leave this page. We're watching. (Not in a creepy way, just in an analytical, data-driven way.)

One Last Thing...

After you leave this page, check the terminal where you're running the server. You'll see the beacon message the browser sent!
